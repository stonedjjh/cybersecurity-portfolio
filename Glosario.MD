# Glossary: Cybersecurity Insights

| Term | Definition |
| :--- | :--- |
| **Adware** | A malicious software that displays unwanted advertisements on a user's device. |
| **Advanced Research Projects Agency Network (ARPANET)** | The first wide-area, packet-switched network with distributed control and one of the first computer networks to implement the TCP/IP protocol suite. |
| **Bot** | An automated software that performs malicious tasks over the internet. The bot malware works alongside other computers infected with the same malware to create a botnet. This botnet is controlled by a command-and-control server responsible for sending out commands. |
| **Bring-your-own-device (BYOD)** | A business policy that permits employees to use personal devices for work-related tasks. |
| **Critical thinking** | A capability of objectively analyzing information and forming reasoned judgments. |
| **Critical thinking model** | A model that comprises five elements: characteristics of thought, technical skills, interpersonal skills, theoretical and experimental knowledge, and intellectual abilities. |
| **Cybersecurity and Infrastructure Security Agency (CISA)** | A United States agency responsible for cybersecurity and infrastructure protection across all levels of government. |
| **Deep fakes** | A technology that simulates voice, image, likeness of an individual. |
| **Department of Homeland Security (DHS)** | A department to safeguard the nation from physical and cyber threats. |
| **Domain name server (DNS)** | A phonebook of the internet, translates domain names to IP addresses so browsers can load internet resources. |
| **Dumpster diving** | An attack that involves searching through an individual's or company's discarded items for valuable data. |
| **Fileless malware** | A form of malicious software that operates within computer memory without leaving any traces on the file system. |
| **Foreign Intelligence Surveillance Court (FISC)** | A foreign intelligence surveillance court entertains applications made by the United States Government for approval of electronic surveillance, physical search, and certain other forms of investigative actions for foreign intelligence purposes. |
| **General data protection regulation (GDPR)** | A regulation that lists the rights of the data subject, meaning the rights of the individuals whose personal data is being processed. |
| **General phishing** | A phishing attack where attackers send emails to a mass group and see who responds. |
| **Hacktivist** | A threat actor who is motivated by political or social agendas. |
| **Hallucination** | Incorrect information given by Gen AI and chatbots. |
| **Hoax** | An attack that involves the distribution of false alarms or misinformation to deceive, create panic, or manipulate individuals. |
| **Impersonation** | An attack where the attacker adopts a false identity to deceive individuals or organizations into granting access to restricted areas or divulging sensitive information. |
| **Insider attack** | An attack attributed to people who have authorized access to an organization's resources, personnel, system, and network. |
| **International Monetary Fund (IMF)** | A policy to promote financial stability and monetary cooperation. |
| **Keylogger** | A malware that records keystrokes on a computer to capture sensitive information. |
| **Logic bomb** | A malicious code that remains dormant until triggered by a specific event or condition. |
| **Malicious insider** | A knowledgeable actor within an entity who exploits their access to inflict damage, steal data, or disrupt operations. |
| **Malvertising** | A malicious advertisement that contains hidden code to deliver malware to users' devices. |
| **Malware** | A malicious software that is specifically created with the intent to disrupt or gain unauthorized access to a system or network. |
| **Mobile device management (MDM)** | A proven methodology and toolset that provides a workforce with mobile productivity tools and applications while keeping corporate data secure. |
| **Mobile telecommunications networks (MTN)** | A telecommunications company. |
| **Multifactor authentication (MFA)** | An account security measure requiring multiple verification forms before granting access to an account or a system. This security measure often combines two or more forms of verification. |
| **National Cyber Security Division (NCSD)** | A division of the Office of Cyber Security and Communications, within the United States Department of Homeland Security's Cybersecurity and Infrastructure Security Agency. |
| **National Security Agency (NSA)** | An agency that prevents and eradicates threats to U.S. national security systems with a focus on the Defense Industrial Base and the improvement of weapons' security. |
| **National Security Decision Directives (NSDD) 145** | A directive that gave the NSA control over all government computer systems containing "sensitive but unclassified" information. |
| **National Telecommunications and Information Systems Security Committee (NTISSC)** | A committee that provides a forum for the discussion of policy issues, sets national policy, and promulgates direction, operational procedures, and guidance for the security of national security systems through the NSTISSC issuance system. |
| **Negligent insider** | An individual who ignores security protocols or bypasses policies for the sake of convenience, due to overconfidence in their knowledge, or sheer disregard for laws. |
| **Oblivious insider** | An unsuspecting employee who, due to lack of awareness or understanding, unintentionally engages in an action that exposes the organization to risk. |
| **Patch** | Security updates on the software. |
| **Penetration testing** | A robust defense strategy, where security specialists simulate cyberattacks to find vulnerabilities before malicious actors do. |
| **Phishing** | A fraudulent attempt to obtain sensitive information by masquerading as a trustworthy entity through electronic communication. |
| **Professional insider** | An individual who has been recruited into an espionage organization. |
| **Quad9** | A free capability to change the DNS in your browser settings to four 9s with a blacklist that prevents visiting bad websites. |
| **Ransomware** | A malware that encrypts files on a victim's system and demands a ransom for their release. |
| **Retrieval augmented generation (RAG)** | A strategy that helps systems become more accurate. |
| **Rootkit** | A stealthy malware that is designed to gain unauthorized access and control over a computer system. |
| **Secure sockets layer (SSL)** | A security measure where data transmitted over the internet is encrypted using SSL certificates from trusted authorities, making it unreadable to anyone who might intercept the communication. |
| **SEO poisoning** | An attack where the attacker puts up a fake website that tricks the search engine into listing it high on the search results. |
| **Shoulder surfing** | An attack involving the direct observation of a person's private information by peering over their shoulder, a practice commonly seen when an individual is entering sensitive data, such as passwords. |
| **Smishing** | An attack where SMS message phishing comes through a search engine. |
| **Social engineering** | A malware attack that hacks humans. |
| **Spam filter** | A critical tool that shields your email inboxes from deceptive tactics. |
| **Spear phishing** | A more targeted version of phishing where attackers tailor their approach to a specific individual or organization. It involves personalized emails that appear credible enough to deceive the target into sharing confidential information. |
| **Spyware** | A malicious software that covertly collects user information and internet behavior without their consent, often for advertising purposes. |
| **Tailgating** | A physical security breach that occurs when an attacker exploits a moment when a controlled entry point is opened by an authorized individual. |
| **Trojan** | A type of malware that disguises itself as legitimate software, deceiving users into executing malicious code that can lead to data theft or system harm. |
| **USA Freedom Act** | An act that was enacted in 2015 and limited the scope of surveillance by intelligence agencies. |
| **USA PATRIOT Act** | An act uniting and strengthening America by providing appropriate tools required to intercept and obstruct terrorism. |
| **Virus** | A malicious software that attaches itself to clean files and spreads through a computer system, corrupting data and hindering performance. |
| **Vishing** | An attack where an attacker uses phone calls instead of emails to scam individuals. |
| **War Operation Plan Response (WOPR)** | A war game programmed to continuously run military simulations and learn over time. |
| **Watering hole attack** | An attack where cybercriminals identify and compromise websites that are popular with their intended victims. |
| **Whaling** | A form of spear phishing that specifically targets high-profile individuals like CEOs or CFOs, the "big fishes" in an organization. |
| **Worm** | A type of malicious software that often spreads autonomously, proliferating its destructive payload without requiring human facilitation. |
| **X-Force** | A global team from IBM comprises ethical hackers, incident responders, researchers, and analysts. |
