# Glosario Unificado de Ciberseguridad

| Término | Definición |
| :--- | :--- |
| **Adware** | Software malicioso que muestra publicidad no deseada en el dispositivo de un usuario. |
| **Agencia de Ciberseguridad y Seguridad de la Infraestructura (CISA)** | Una agencia de Estados Unidos responsable de la ciberseguridad y de la protección de infraestructuras en todos los niveles de gobierno. |
| **Agencia de Seguridad Nacional (NSA)** | Una agencia que previene y erradica amenazas a los sistemas de seguridad nacional de EE. UU., con enfoque en la Base Industrial de Defensa y en la mejora de la seguridad de los armamentos. |
| **Ataque de suplantación (Impersonation)** | Un ataque donde el atacante adopta una identidad falsa para engañar a individuos u organizaciones y obtener acceso a áreas restringidas o divulgar información sensible. |
| **Ataque de 'watering hole'** | Un ataque donde los cibercriminales identifican y comprometen sitios web que son populares entre sus víctimas previstas. |
| **Ataque de 'whaling'** | Una forma de spear phishing que se dirige específicamente a individuos de alto perfil como CEOs o CFOs, los "peces gordos" en una organización. |
| **Ataque interno (Insider attack)** | Un ataque atribuido a personas que tienen acceso autorizado a los recursos, personal, sistemas y red de una organización. |
| **Bring-your-own-device (BYOD)** | Una política empresarial que permite a los empleados usar dispositivos personales para tareas relacionadas con el trabajo. |
| **Bulo (Hoax)** | Un ataque que implica la distribución de falsas alarmas o desinformación para engañar, crear pánico o manipular a individuos. |
| **Ciberactivista (Hacktivist)** | Un actor de amenaza motivado por agendas políticas o sociales. |
| **Comité de Seguridad de Sistemas Nacionales de Telecomunicaciones e Información (NTISSC)** | Un comité que ofrece un foro para la discusión de cuestiones de política, establece la política nacional y promulga directrices, procedimientos operativos y orientaciones para la seguridad de los sistemas de seguridad nacional. |
| **Copia de pantalla (Shoulder surfing)** | Un ataque que consiste en la observación directa de la información privada de una persona mirando por encima de su hombro, práctica común al ingresar datos sensibles, como contraseñas. |
| **Deep fakes** | Tecnología que simula la voz, imagen y apariencia de un individuo. |
| **Departamento de Seguridad Nacional (DHS)** | Un departamento encargado de proteger la nación contra amenazas físicas y cibernéticas. |
| **Directiva de Decisión de Seguridad Nacional (NSDD) 145** | Una directiva que otorgó a la NSA control sobre todos los sistemas informáticos gubernamentales que contienen información "sensible pero no clasificada". |
| **División Nacional de Ciberseguridad (NCSD)** | Una división dentro del Departamento de Seguridad Nacional de Estados Unidos. |
| **DNS (Domain Name Server)** | Un "directorio telefónico" de internet que traduce nombres de dominio a direcciones IP para que los navegadores puedan cargar recursos en línea. |
| **Dumpster diving** | Un ataque que implica buscar entre los elementos desechados de un individuo o empresa en busca de datos valiosos. |
| **Error (Hallucination)** | Información incorrecta proporcionada por la IA generativa y los chatbots. |
| **Filtro de spam** | Una herramienta crítica que protege las bandejas de entrada de correo electrónico de tácticas engañosas. |
| **Fondo Monetario Internacional (IMF)** | Una política destinada a promover la estabilidad financiera y la cooperación monetaria. |
| **Generación Aumentada por Recuperación (RAG)** | Una estrategia que ayuda a los sistemas a ser más precisos. |
| **Ingeniería social (Social engineering)** | Un ataque de malware que se dirige a humanos (manipulación psicológica). |
| **Insider malicioso (Malicious insider)** | Un actor con conocimiento dentro de una entidad que explota su acceso para infligir daño, robar datos o interrumpir operaciones. |
| **Insider negligente (Negligent insider)** | Un individuo que ignora los protocolos de seguridad o evade políticas por conveniencia, exceso de confianza o simple negligencia. |
| **Insider despistado (Oblivious insider)** | Un empleado desprevenido que, por falta de concienciación o comprensión, realiza involuntariamente una acción que expone a la organización a riesgos. |
| **Insider profesional (Professional insider)** | Un individuo que ha sido reclutado por una organización de espionaje. |
| **Keylogger** | Un malware que registra las pulsaciones de teclado en una computadora para capturar información sensible. |
| **Ley de Libertad de EE. UU. (USA Freedom Act)** | Una ley promulgada en 2015 que limitó el alcance de la vigilancia por parte de las agencias de inteligencia. |
| **Ley PATRIOT de EE. UU. (USA PATRIOT Act)** | Una ley destinada a unir y fortalecer a Estados Unidos proporcionando las herramientas apropiadas necesarias para interceptar y obstaculizar el terrorismo. |
| **Malware** | Software malicioso creado específicamente con la intención de interrumpir u obtener acceso no autorizado a un sistema o red. |
| **Malware sin archivo (Fileless malware)** | Una forma de software malicioso que opera dentro de la memoria de la computadora sin dejar rastros en el sistema de archivos. |
| **Malvertising** | Publicidad maliciosa que contiene código oculto para entregar malware a los dispositivos de los usuarios. |
| **Mobile device management (MDM)** | Una metodología y conjunto de herramientas probados que proporcionan a la plantilla herramientas y aplicaciones móviles de productividad, manteniendo seguros los datos corporativos. |
| **Modelo de pensamiento crítico** | Un modelo que comprende cinco elementos: características del pensamiento, habilidades técnicas, habilidades interpersonales, conocimientos teóricos y experimentales, y capacidades intelectuales. |
| **Multifactor authentication (MFA)** | Una medida de seguridad de cuenta que requiere múltiples formas de verificación antes de otorgar acceso a una cuenta o sistema. |
| **Parche (Patch)** | Actualizaciones de seguridad en el software. |
| **Pensamiento crítico** | La capacidad de analizar objetivamente la información y formar juicios razonados. |
| **Phishing** | Intento fraudulento de obtener información sensible haciéndose pasar por una entidad de confianza a través de comunicación electrónica. |
| **Phishing general** | Un ataque de phishing donde los atacantes envían correos electrónicos a un grupo masivo y esperan a ver quién responde. |
| **Phishing por SMS (Smishing)** | Un ataque de phishing por mensaje de texto. (Nota: tu definición original decía "a través de un motor de búsqueda", que no es estándar para smishing. Se corrigió a "mensaje de texto"). |
| **Phishing dirigido (Spear phishing)** | Versión más dirigida del phishing donde los atacantes adaptan su enfoque a un individuo u organización específicos. |
| **Phishing por voz (Vishing)** | Un ataque donde un atacante utiliza llamadas telefónicas en lugar de correos electrónicos para estafar a individuos. |
| **Prueba de penetración (Penetration testing)** | Una sólida estrategia de defensa donde especialistas en seguridad simulan ciberataques para encontrar vulnerabilidades antes que los actores maliciosos. |
| **Quad9** | Una capacidad gratuita para cambiar el DNS en la configuración de su navegador a cuatro 9s con una lista negra que evita visitar sitios web maliciosos. |
| **Ransomware** | Un malware que cifra archivos en el sistema de una víctima y exige un rescate por su liberación. |
| **Red de la Agencia de Proyectos de Investigación Avanzada (ARPANET)** | La primera red de área amplia con conmutación de paquetes y control distribuido, y una de las primeras redes informáticas en implementar la suite de protocolos TCP/IP. |
| **Redes de telecomunicaciones móviles (MTN)** | Una empresa de telecomunicaciones. |
| **Reglamento General de Protección de Datos (GDPR)** | Un reglamento que enumera los derechos del titular de los datos, es decir, los derechos de las personas cuyos datos personales están siendo procesados. |
| **Rootkit** | Malware sigiloso diseñado para obtener acceso no autorizado y control sobre un sistema informático. |
| **Secure Sockets Layer (SSL)** | Una medida de seguridad donde los datos transmitidos por internet se cifran mediante certificados SSL de autoridades de confianza. |
| **SEO Poisoning** | Un ataque donde el atacante crea un sitio web falso que engaña al motor de búsqueda para que lo liste en una posición alta en los resultados. |
| **Software espía (Spyware)** | Software malicioso que recopila de forma encubierta información del usuario y comportamiento en internet sin su consentimiento. |
| **Tailgating** | Un fallo de seguridad física que ocurre cuando un atacante aprovecha un momento en que un punto de entrada controlado es abierto por un individuo autorizado. |
| **Bomba lógica (Logic bomb)** | Código malicioso que permanece inactivo hasta que es activado por un evento o condición específica. |
| **Bot** | Software automatizado que realiza tareas maliciosas en internet. Funciona junto a otras computadoras infectadas para crear una botnet. |
| **Caballo de Troya (Trojan)** | Un tipo de malware que se disfraza como software legítimo, engañando a los usuarios para que ejecuten código malicioso. |
| **Tribunal de Vigilancia de Inteligencia Extranjera (FISC)** | Un tribunal que atiende solicitudes de vigilancia electrónica y registros físicos con fines de inteligencia extranjera. |
| **Virus** | Software malicioso que se adhiere a archivos limpios y se propaga, corrompiendo datos y obstaculizando el rendimiento. |
| **War Operation Plan Response (WOPR)** | Un "war game" programado para ejecutar continuamente simulaciones militares y aprender con el tiempo. |
| **Gusano (Worm)** | Un tipo de software malicioso que a menudo se propaga de forma autónoma, proliferando su carga destructiva sin requerir facilitación humana. |
| **X-Force** | Un equipo global de IBM compuesto por hackers éticos, respondedores de incidentes, investigadores y analistas.|
